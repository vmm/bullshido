# 🥋 Bullshido

**The Way of the McDojo** — A side-scrolling beat 'em up where your martial arts moves are too powerful.

Flick a finger and enemies fly off screen. Poke a toe and they go orbital. Your Bullshido cannot be contained.

## Play

```bash
npx serve . -l 3333
# Open http://localhost:3333
```

## Controls

| Key | Move | Effect |
|-----|------|--------|
| `← →` / `A D` | Move | Walk left/right |
| `Z` | **The Flick** | Finger flick — enemies fly horizontally |
| `X` | **Ki Shout** | Shockwave pushes all nearby enemies |
| `C` | **No-Touch KO** | Wave hand — enemy collapses dramatically |
| `V` | **Slow-Mo Palm** | Gentle push — enemy backflips off-screen |
| `B` | **Lazy Sweep** | Barely lifts foot — enemies trip into oblivion |
| `N` | **Toe Poke of Doom** | One toe. Straight to orbit. |

## Tech

Pure HTML5 Canvas + vanilla JavaScript. No frameworks, no build step, no dependencies.
